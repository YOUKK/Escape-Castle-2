# Escape Castle

* 깜깜한 미로 속에서 몬스터를 피해 열쇠를 얻고 탈출하는 게임입니다.
* 시간 제한 안에 탈출해야 합니다.
* 2종의 몬스터가 있습니다.
* 총 3개의 스테이지로 구성되어있습니다.
* 스테이지는 점점 난이도가 높아지며, 맵 크기 또한 점점 커집니다.
